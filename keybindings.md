# Keybindings

## File navigation

- <]i> jump to respective border line top
- <]i> jump to respective border line bottom>

## Visual Keybindings

- <vai> - object scope with border
- <vii> - object scope
- <v.> - smart scope
- <v;> - container outer
- <vi> - container inner
- <v,> - previous selection
